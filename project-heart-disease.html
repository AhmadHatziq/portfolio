Binary Heart Classification

The aim of this project is to give a rough estimate for individuals to check if they are at risk of heart disease.

Summary

This project is part of the NUS module CS3244: Machine Learning.

The dataset is obtained from the UCL Machine Learning repository. The aim is to create a model that effectively classifies heart disease (Yes or No) based on AUC Score. The ML models are trained on a 75-25 train-test split with 5-fold cross validation.

The best model obtained was by using LDA. To further investigate the variables, Partial Dependance Plots (PDP) for each non-categorical variable were obtained. The reference notebook for the PDP is here. It was found that Serum Cholestrol was most significant in affecting heart disease, based on the UCL dataset.
